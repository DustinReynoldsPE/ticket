# TUI: detail view and navigation

Build the ticket detail view with full ticket display and back navigation.

## Design

Files: internal/tui/detail.go
Approach:
- detail.go: ticket detail model displaying full ticket content
  - Render frontmatter fields in a formatted header block
  - Render markdown body sections (Description, Design, Acceptance, Notes)
  - Scrollable viewport for long tickets (bubbles/viewport)
  - esc to return to list view
  - j/k or arrow keys to scroll
  - q to quit from any view
- Styling: section headers highlighted, metadata in key:value format
- Dep/link IDs shown as part of frontmatter display

## Acceptance Criteria

Enter on list item shows full ticket. Esc returns to list. Long tickets scroll.

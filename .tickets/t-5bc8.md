# GoReleaser and packaging setup

Configure GoReleaser for cross-platform builds and update Homebrew/AUR packaging.

## Design

Files: .goreleaser.yml, .github/workflows/release.yml, pkg/aur/PKGBUILD
Approach:
- .goreleaser.yml: builds for darwin/arm64, darwin/amd64, linux/arm64, linux/amd64. Binary name: tk. Archive format: tar.gz.
- GoReleaser homebrew tap support: auto-update wedow/homebrew-tools formula
- Update release.yml: replace manual build steps with goreleaser action
- AUR PKGBUILD: update to install Go binary, drop bash/coreutils/awk deps, remove jq optdep
- Checksums generated by GoReleaser automatically

## Acceptance Criteria

goreleaser release --snapshot builds all 4 platform binaries. Release workflow produces GitHub release with binaries.

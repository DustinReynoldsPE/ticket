# TUI: app scaffold and list view

Build the bubbletea TUI entry point and interactive ticket list view.

## Design

Files: cmd/ui.go, internal/tui/app.go, internal/tui/list.go
Approach:
- cmd/ui.go: tk ui command, initializes bubbletea program with app model
- app.go: top-level model managing view state (list vs detail), shared store reference
- list.go: ticket list view using bubbles/list component
  - Columns: priority, type, ID, title (matching tk ls format)
  - Arrow key navigation
  - / to enter filter mode (text input filters title)
  - Tab to cycle status filter tabs (all, open, in_progress, closed)
  - Enter to switch to detail view
  - q to quit
- Styling via lipgloss: subtle colors for priority, type badges
- Load tickets from FileStore on startup and on view return

## Acceptance Criteria

tk ui launches, displays ticket list, arrow keys navigate, q quits

# CLI commands: dep, undep, link, unlink

Implement dependency and link management CLI commands.

## Design

Files: cmd/dep.go, cmd/link.go
Approach:
- dep id dep-id: add dep-id to ticket id's deps array, update file
- dep tree [--full] id: display dependency tree using DepTree from core
- dep cycle: find and display cycles using FindCycles from core
- undep id dep-id: remove dep-id from ticket id's deps array
- link id1 id2 [id3...]: add symmetric links between all specified tickets
- unlink id target-id: remove link between two tickets
- All commands validate ticket IDs exist before modifying
- Trigger status propagation after dep changes where relevant

## Acceptance Criteria

Dep add/remove/tree/cycle and link/unlink match bash behavior. Test suite dependency assertions pass.
